Program Vision

단순한 Houdini 사용자가 아닌, 재사용 가능한 프로시저럴 시스템과 파이프라인 툴을 설계하는 Technical Artist를 육성합니다. 아티스트의 창의성과 엔지니어의 사고방식을 결합한 차세대 프로덕션 워크플로우를 경험하세요.

Systems Thinking

Program Structure

16주 커리큘럼

4개 Phase로 기초 원리부터 실전 툴셋 포트폴리오까지 단계적으로 완성합니다.

PHASE 01 Weeks 1–4

Procedural Foundations

Houdini 노드 기반 사고방식과 프로시저럴 모델링의 핵심 원리를 익힙니다.

  • SOP 네트워크 구조, 파라미터 인터페이스, 노드 그래프 설계
  • Attributes, Groups, VEX 기초 – 포인트 조작 및 데이터 흐름 이해
  • Scatter & Instancing, Copy to Points, 랜덤 변형 시스템
  • 프로젝트: 모듈러 프롭 킷 (Procedural Prop Pack)
PHASE 02 Weeks 5–8

Procedural Systems & HDA Design

재사용 가능한 HDA 툴을 설계하고 복잡한 프로시저럴 시스템을 구축합니다.

  • Houdini Digital Asset (HDA) 제작, 파라미터 UI, 잠금 및 배포
  • L-System, Growth Simulation, 유기적 형태 생성
  • Terrain & Heightfield 제작, 침식 시뮬레이션, 레이어 마스크
  • 프로젝트: 커스텀 HDA 툴팩 (Building Generator / Terrain HDA)
PHASE 03 Weeks 9–12

Real-Time Pipeline & Unreal Integration

Houdini Engine과 PCG를 활용한 실시간 프로시저럴 파이프라인을 구현합니다.

  • Houdini Engine for Unreal – HDA 실시간 업데이트 및 파라미터 제어
  • Unreal PCG Graph – 규칙 기반 월드 배치 시스템 설계
  • Procedural Biome / City Block / Modular Structure 자동 생성
  • 프로젝트: Procedural Environment Demo (Unreal + Houdini Hybrid)
PHASE 04 Weeks 13–16

Portfolio Production & Career Prep

최종 캡스톤 프로젝트 완성 및 산업 현장 수준의 포트폴리오를 준비합니다.

  • 캡스톤 프로젝트: 프로시저럴 월드 툴셋 또는 HDA 파이프라인 시스템
  • Breakdown 영상 제작, 프로세스 문서화, 포트폴리오 패키징
  • 산업 멘토 크리틱, 최종 발표 및 커리어 브랜딩
  • 최종 산출물: Procedural Toolset + Real-Time Environment + Demo Reel
Specialization Modules

핵심 학습 모듈

각 모듈은 실무 직무에 직접 연결되는 기술 스택을 중심으로 구성됩니다. 프로시저럴 모델링, 툴 개발, 리얼타임 월드 빌딩 세 축을 깊이 있게 다룹니다.

Procedural Modeling & VEX

파라미터 기반 모델링 시스템과 VEX 스크립팅으로 정밀 제어 가능한 지오메트리를 생성합니다.

  • SOP 네트워크 심화, For-Each 루프, 다중 입력 처리
  • VEX 함수, 벡터 수학, 노이즈 함수 활용
  • 포트폴리오: 파라미터 제어 가능한 Modular Asset System

HDA Tool Development

팀 공유 가능한 고품질 HDA를 설계하고, 프로덕션 파이프라인에 통합하는 능력을 키웁니다.

  • HDA 파라미터 UI/UX 설계, Python 스크립팅, 콜백 함수
  • Building Generator, Road Network, Scatter System HDA
  • 포트폴리오: 3–5개 배포 가능한 프로덕션 HDA 패키지

Real-Time World Generation

Houdini Engine과 Unreal PCG를 결합한 대규모 프로시저럴 환경 생성 파이프라인을 구축합니다.

  • PCG Graph 노드 설계, 필터링, 계층형 월드 빌딩
  • Houdini-Unreal 하이브리드 워크플로우, Nanite & Lumen 연동
  • 포트폴리오: Playable Procedural Environment + PCG Demo Level
