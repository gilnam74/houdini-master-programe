Houdini Master Program – Film & Game (DOOBOB)

Think Tank형 성장 곡선에 DOOBOB의 실시간 제작 역량을 결합한 12개월 집중 과정. Houdini 기반 Procedural FX를 Film, Game, Tech Art production으로 연결합니다.

12 Months48–52 weeks
3 TracksFilm / Game / Tech Art
PipelineHoudini → Unreal / Niagara / PCG
OutcomeDemo Reel + Portfolio + Studio Workflow

Program Vision

Houdini Specialist를 넘어 Real-Time과 Pipeline을 이해하는 Production-ready artist를 육성합니다. 핵심 차별 요소는 FX + Environment + Tech Art 융합, Mentor 중심 피드백, Studio-style 팀 제작입니다.

Procedural Mindset

노드/속성/시스템 관점으로 재사용 가능한 디자인 구조를 설계합니다.

Real-Time Integration

Houdini 결과물을 Unreal, Niagara, PCG 파이프라인으로 연결합니다.

Studio Simulation

실무처럼 역할 분화된 팀 스프린트와 산업 멘토 크리틱으로 완성도를 높입니다.

60% Base · Ivory Mist #FBF8F2 · Main background and large surfaces
30% Core · Deep Teal #1F5E66 · Brand blocks and hierarchy anchors
10% Accent · Copper Clay #B86C42 · Emphasis and action points

보조 톤은 Slate Ink(#1D2328), Blue Steel(#345C86), Sage Dust(#7A8F6A)로 구성해 대비와 안정감을 동시에 확보했습니다.

Overall Structure

총 4개 Phase로 기초부터 포트폴리오 제작까지 연결됩니다.

Phase 1

Procedural Foundations

12 weeks · Houdini 사고방식과 노드 기반 제작

  • Houdini Fundamentals, Procedural Modeling, Attributes & VEX
  • Lookdev & Lighting, Intro FX, USD Scene Structure
  • Project: Procedural Asset Pack / Parametric Props / Modular Kit
Phase 2

FX & Simulation Specialist

12 weeks · 시뮬레이션 핵심 능력 집중

  • Particles, RBD, Pyro, FLIP, Vellum, Groom
  • FX Optimization, Rendering & Caching
  • Project: FX Shot (Film) + FX Asset (Game)
Phase 3

Track Specialization

12 weeks · Film / Game / Procedural Tech Art 트랙 심화

  • 목표 직무 기반 모듈 선택
  • 트랙별 대표 산출물 제작
  • 캡스톤 초안 및 실무형 리뷰
Phase 4

Master Production

12–16 weeks · 포트폴리오와 실전 제작

  • Industry mentor pairing, Studio sprint
  • Demo reel, Career branding, Interview prep
  • Final: AAA FX shots / Real-time system / Procedural tools

Phase 3 Tracks

트랙마다 결과물의 기준이 다릅니다. Film은 샷 완성도, Game은 실시간 최적화, Tech Art는 툴/파이프라인 재사용성이 핵심입니다.

Film FX Track

고품질 시뮬레이션 샷을 설계, 라이팅, 렌더, 컴포지팅까지 완성.

  • Hero Pyro/FLIP/RBD/Vellum 시뮬레이션
  • Solaris/USD, Karma, AOVs, Nuke comp
  • 포트폴리오: Film FX Shots 2–3 + Breakdown Package

Game FX Track

게임 플레이 환경에서 안정적으로 동작하는 리얼타임 FX 제작.

  • Niagara 시스템, 이벤트, 리본, GPU/CPU
  • Shader/Material 기반 stylized FX
  • 포트폴리오: FX Pack 10~20 + Playable Showcase + Budget Sheet

Procedural Tech Art Track

아티스트와 레벨팀을 위한 프로시저럴 툴/월드 시스템 구축.

  • HDA 설계/배포, Houdini Engine for Unreal
  • PCG Graph + Houdini Hybrid Workflow
  • 포트폴리오: Environment Toolset + UE PCG Demo
