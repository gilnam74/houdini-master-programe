Houdini Master Program
Film & Game

안녕하세요. 저는 DNEG, ILM, Weta, MPC 등의 헐리우드 VFX 스튜디오에서 수십 편의 영화에 참여했습니다. 최근까지 DNEG의 Effects Supervisor로 근무했고, 캐나다 대학교에서 강의했으며, 현재는 미국 LA의 게임회사에서 Technical Art Director로 일하고 있습니다.

이 코스는 20여 년의 실무와 강의 경험을 바탕으로 FX 아티스트에게 꼭 필요한 실무 지식, 포트폴리오 구성 방법, 그리고 스스로 공부하는 방법까지 익히도록 설계되었습니다. 또한 게임 제작에 후디니를 적용하려는 아티스트에게도 직접적인 도움이 되도록 구성했습니다.

12 Modules

커리큘럼

후디니 FX의 기초 이해부터 포트폴리오용 프로젝트까지, 12개의 모듈로 단계적으로 연결됩니다.

Module 01

파티클을 내가 원하는 대로 컨트롤하기

  • Houdini 인터페이스 / 좋은 레퍼런스 찾기 / POP
  • 빠른 오브젝트 대응 / Solver / Collision 이해
  • Newtonian Physics 소개

Module 02

Introduction to Math in Houdini

  • 필요한 수학 함수 공부하기
  • Vector Addition / Subtraction / Multiplication
  • Dot Product / Cross Product

Module 03

VEX

  • Infection Solver 만들기
  • Attribute Wrangle 사용하기

Module 04

Disintegration Project

  • Emit Particles
  • Growth Effect (Infection Solver 응용)
  • Vellum Cloth

Module 05

RBD Destruction 건물 부수기 I

  • 디스트럭션 계획 전략 수립
  • Fracturing / RBD Bullet Solver / Constraint

Module 06

RBD Destruction 건물 부수기 II

  • Secondary Particles / Volumes
  • Rendering

Module 07

불타는 캐릭터: Sticky Fire Advanced Technique

  • Smoke Sourcing
  • Pyro Simulation
  • Rendering

Module 08

물속 캐릭터 움직임에 따른 이펙트

  • 바닷속 장면에 필요한 요소들
  • Bubble / Blood 셋업

Module 09

Procedural Effect (VFX와 게임)

  • Frozen 이펙트 따라하기
  • Ground Deformation 이펙트 따라하기

Module 10

Zombie Burning / Zombie Explode

  • Growth Effect (Infection Solver 활용)
  • RBD 응용

Module 11

영화 '트랜스포머' Ground Interaction 따라하기

  • Ground Interaction 공식의 이해
  • Dot Product, Reflect 활용
  • 셋업의 무한 응용

Module 12

베놈 변신 이펙트 따라하기 (VFX와 게임)

  • Procedural Trail 셋업
  • POP Fluid / VDB Post Process
  • UV Transfer
