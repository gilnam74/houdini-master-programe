Program Structure

16-Week Curriculum

Four structured phases take you from core procedural principles to a production-ready toolset portfolio.

PHASE 01 Weeks 1–4

Procedural Foundations

Learn the core principles of Houdini's node-based thinking and procedural modeling.

  • SOP network structure, parameter interfaces, and node graph design
  • Attributes, Groups, VEX fundamentals — point manipulation and data flow
  • Scatter & Instancing, Copy to Points, randomized variation systems
  • Project: Modular Prop Pack (Procedural Prop System)
PHASE 02 Weeks 5–8

Procedural Systems & HDA Design

Design reusable HDA tools and build complex procedural generation systems.

  • HDA creation, parameter UI/UX design, locking, and deployment
  • L-Systems, growth simulations, and organic form generation
  • Terrain & Heightfield generation, erosion simulation, layer masking
  • Project: Custom HDA Tool Pack (Building Generator / Terrain HDA)
PHASE 03 Weeks 9–12

Real-Time Pipeline & Unreal Integration

Build a real-time procedural pipeline using Houdini Engine and Unreal PCG.

  • Houdini Engine for Unreal — live HDA updates and parameter control in-editor
  • Unreal PCG Graph — rule-based world placement system design
  • Procedural Biome / City Block / Modular Structure auto-generation
  • Project: Procedural Environment Demo (Unreal + Houdini Hybrid)
PHASE 04 Weeks 13–16

Portfolio Production & Career Prep

Complete your capstone project and prepare an industry-standard portfolio.

  • Capstone: Procedural World Toolset or HDA Pipeline System
  • Breakdown video production, process documentation, portfolio packaging
  • Industry mentor critique, final presentation, and career branding
  • Final deliverables: Procedural Toolset + Real-Time Environment + Demo Reel
Specialization Modules

Core Learning Modules

Each module is built around a technical skill stack that maps directly to production roles. The program covers three pillars: procedural modeling, tool development, and real-time world building.

Procedural Modeling & VEX

Generate precisely controlled geometry using parameter-driven modeling systems and VEX scripting.

  • Advanced SOP networks, For-Each loops, multi-input processing
  • VEX functions, vector math, and noise function workflows
  • Portfolio: Parameter-controllable Modular Asset System

HDA Tool Development

Design high-quality, team-deployable HDAs and integrate them into production pipelines.

  • HDA parameter UI/UX design, Python scripting, callback functions
  • Building Generator, Road Network, Scatter System HDAs
  • Portfolio: 3–5 production-ready deployable HDA packages

Real-Time World Generation

Build large-scale procedural environment pipelines combining Houdini Engine and Unreal PCG.

  • PCG Graph node design, filtering, and hierarchical world building
  • Houdini-Unreal hybrid workflow, Nanite & Lumen integration
  • Portfolio: Playable Procedural Environment + PCG Demo Level
